# weather-the-storm

## Overview

I wrote a basic weather API request program that writes the response to a file. It also can read from that file, or others if you've made other files before.

My purpose in creating this was to make an actual project in Java using Object-Oriented Programming principles. I had learn the basics of programming in Java before, but I never actually made a project with classes before. 

[Software Demo Video](https://youtu.be/Xio4cFlX-7s)

# Development Environment

Used EclipseIDE to build the program.

I used Java. I used the java.util.Scanner, java.io.File, java.io.FileWriter, java.io.FileReader, java.io.BuffWriter, java.io.BuffWriter, java.io.IOException, java.InputStreamReader, java.net.HttpUrlConnection, and java.net.URL libraries.
